大涼銀行（大涼帝國官方金融系統）
 
項目簡介
 
本項目是模擬大涼帝國官方銀行的網頁應用，基於前端技術實現完整的個人金融服務場景，包含賬戶管理、轉帳匯款、交易查詢、理財產品等核心功能，整體設計融合傳統中式華美風格與現代銀行排版邏輯，貨幣單位統一使用「盧尼茲」，文字全程採用繁體中文呈現。
 
核心功能模塊
 
模塊名稱 對應頁面 核心功能描述 
用戶認證  index.html  賬戶開戶、登錄驗證，未登錄自動跳轉，登錄後保存用戶狀態 
個人金融中心  bank-home.html  餘額展示、常用功能導航、最近交易記錄、銀行公告與金融資訊瀏覽 
轉帳匯款  bank-transfer.html  境內轉帳、跨行匯款、收款人管理、轉帳記錄查詢 
帳單查詢  bank-bill.html  歷史交易明細查詢、交易類型篩選、時間區間查詢、交易憑證下載 
充值儲值  bank-recharge.html  線下櫃檯充值、第三方支付充值、充值記錄查詢 
提現兌換  bank-withdraw.html  盧尼茲提現、外幣兌換、提現記錄與匯率查詢 
財富管理  bank-wealth.html  理財產品展示、產品詳情查詢、投資認購、收益明細查詢 
客戶服務  bank-service.html  常見問題查詢、在線咨詢入口、意見反饋、銀行網點查詢 
賬戶詳情  bank-account.html  個人信息管理、賬戶安全設置、實名認證補錄、綁定銀行卡管理 
 
設計特色
 
1. 視覺風格：以朱紅色（#8B0000）與鎏金色（#D4AF37）為主色調，背景疊加祥雲、纏枝蓮、回紋等中式暗紋，模塊邊框搭配角標、裝飾線等傳統元素，避免純色單調，凸顯華美質感。
2. 排版邏輯：參照國內主流銀行網頁布局，頂部固定導航欄、中間分區展示核心功能與數據、底部補充資訊與版權，符合用戶使用習慣。
3. 交互體驗：按鈕懸停效果、表格行高亮、鏈接點擊反饋等微交互設計，功能模塊分區明確，導航層級清晰，支持多頁面無縫跳轉。
 
技術依賴
 
- 前端基礎：HTML5 + CSS3 + JavaScript
- 存儲方案：localStorage（用戶數據、交易記錄、賬戶餘額等本地存儲）
- 兼容環境：現代瀏覽器（Chrome、Firefox、Edge等），響應式設計支持移動端與PC端訪問
 
快速入門
 
1. 環境準備
 
- 無需後端服務器，直接下載項目所有HTML文件至本地文件夾
- 確保文件間路徑正確（所有頁面放在同一目錄下）
 
2. 使用流程
 
1. 打開  index.html  進入登錄頁，輸入任意賬戶名與密碼（首次輸入即自動創建賬戶，初始餘額10000盧尼茲）
2. 登錄後自動跳轉至  bank-home.html （個人金融中心），可通過頂部導航欄或中間功能按鈕進入對應模塊
3. 所有操作數據實時保存至本地，刷新頁面後數據不丟失
4. 退出登錄可點擊頂部導航欄「安全登出」按鈕
 
3. 測試數據說明
 
- 初始賬戶餘額：10000盧尼茲（開戶贈送）
- 默認交易記錄：包含開戶贈送、理財收益、轉帳支出、充值到賬、生活繳費5條模擬數據
- 理財產品：默認展示「國祚理財」產品（預期年化5.2%）
 
注意事項
 
1. 本項目僅為前端模擬，所有數據存儲於本地localStorage，清除瀏覽器數據後將丟失
2. 不支持真實金融交易，僅用於場景模擬與展示
3. 頁面間依賴相對路徑，請勿隨意修改文件目錄結構
4. 若需擴展功能（如後端聯動、數據持久化），可基於現有前端架構對接後端接口
 
擴展建議
 
1. 後端集成：對接Node.js/Java後端服務，實現數據庫存儲與用戶認證
2. 功能擴充：增加信用卡管理、貸款申請、保險產品、線上櫃檯等模塊
3. 體驗優化：加入動畫過渡效果、加載狀態提示、錯誤處理反饋
4. 安全升級：增加密碼加密存儲、驗證碼登錄、交易密碼二次驗證等功能
